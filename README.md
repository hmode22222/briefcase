# Briefcase - Demo Android app

تطبيق أندرويد تجريبي بسيط (Kotlin) يضم شاشة واحدة مع عداد نقرات.

ملف المشروع مدمج لتستطيع فتحه في Android Studio وبناء APK محليًا.

لتجميع APK محليًا:
1. افتح المشروع في Android Studio.
2. Build -> Build Bundle(s) / APK(s) -> Build APK(s)

أو من سطر الأوامر (لـ Unix/macOS):

./gradlew assembleDebug

الـ APK الناتج (debug) سيكون في: app/build/outputs/apk/debug/app-debug.apk
